\section{Background}
\label{sec:background}

\paragraph{Marked temporal point processes.}
An MTPP models a sequence $\mathcal{H}_T=\{(t_i,k_i)\}_{i=1}^{N_T}$ of events with
times $t_i$ and types $k_i\in\{1,\dots,K\}$ through per-type conditional intensities
$\lambda_k^*(t)$. Writing $\lambda^*(t)=\sum_k\lambda_k^*(t)$ for the total (ground)
intensity and $p^*(k\mid t)=\lambda_k^*(t)/\lambda^*(t)$ for the mark distribution,
the log-likelihood of a sequence is
\[
\log\mathcal{L}
=\sum_i \Bigl[\log\lambda^*(t_i)+\log p^*(k_i\mid t_i)\Bigr]
-\int_0^T\!\lambda^*(s)\,ds ,
\]
the integral being the \emph{compensator}. Sampling uses Ogata thinning, which
requires a dominating rate $\bar\lambda\ge\lambda^*(t)$; a model whose intensity has
no computable upper bound must resort to heuristic (and bias-prone) dominating-rate
estimates. Classical self-exciting intensities are Hawkes processes
\citep{Hawkes1971}, widely used for LOB event flow
\citep{Jain2024LimitProcess,Morariu-Patrichi2022State-dependentModelling}; neural
versions replace the parametric kernel with recurrent
\citep{du2016recurrent,Mei2017TheProcess} or attention-based \citep{zhang2020self}
history encoders.

\paragraph{The \sppp{} backbone.}
\sppp{} \citep{Shi2022StateSimulation} is a state-space point process for LOB
streams: $L$ stacked latent-linear-Hawkes/SSM layers with diagonal state,
ZOH-discretized between events and computable by parallel scan,
\[
x^{(\ell)}_{t'}=\bar A^{(\ell)} x^{(\ell)}_{t}
+(\bar A^{(\ell)}\!-\!I)\,\tilde B^{(\ell)} u^{(\ell-1)}
\;\bigl[+\,\tilde E^{(\ell)}\alpha\ \text{at an event}\bigr],
\]
with a LayerNorm'd residual readout
$u^{(\ell)}=\mathrm{LayerNorm}\bigl(\mathrm{GELU}(y^{(\ell)})+u^{(\ell-1)}\bigr)$
producing the stack output $u(t):=u^{(L)}(t)\in\mathbb{R}^{H}$. The published model
reads the rate from $u(t)$ through a projection-and-softplus head,
$\lambda = s\cdot\mathrm{softplus}(w^{\!\top}u+b)$, which is unbounded above; marks
are coupled to the same rate field. \ssppp{} keeps everything up to and including
$u(t)$ and changes only these two heads.

\paragraph{Evaluation axes.}
We score \emph{prediction} per genuine test event --- the time/mark NLL split, type
accuracy and perplexity, time-MAE, and time-rescaling calibration (rescaled
inter-event masses should be Exp(1); we report their KS distance and mean) --- and
\emph{simulation} by 600\,s closed-loop roll-outs bucketed into 1\,s bins, compared
to real data on event rate, Fano factor, volatility clustering, and return
autocorrelation \citep{cont2001empirical,vyetrenko2020getreal}.
